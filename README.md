# Guía de repaso de los proyectos

Esta carpeta reúne un README por proyecto para estudiar el código con más detalle, línea por línea o por bloques pequeños.

## Proyectos incluidos

- [Laboratorio0](Laboratorio0/README.md)
- [lab2](lab2/README.md)
- [lab4](lab4/README.md)
- [labDataBase](labDataBase/README.md)
- [formulario](formulario/README.md)
- [ejemplo1](ejemplo1/README.md)
- [MyApplication](MyApplication/README.md)
- [MyApplication2](MyApplication2/README.md)
- [database](database/README.md)

## Cómo usar esta guía

1. Abre el README del proyecto que quieras repasar.
2. Lee primero el resumen para entender la idea general.
3. Después revisa las líneas explicadas para memorizar qué hace cada parte del código.

## Nota

Los proyectos usan estructuras Android parecidas, pero practican temas distintos: pantallas básicas, persistencia interna, almacenamiento externo, formularios, envío de datos entre actividades y CRUD con SQLite.
